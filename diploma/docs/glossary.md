# Глоссарий — согласованная терминология

> **Правило для агента:** при упоминании любого термина из этого списка использовать **только утверждённый русский эквивалент** (или принятую транслитерацию). Не варьировать в разных секциях.
>
> При первом упоминании в работе — полное русское название + сокращение или английский термин в скобках, далее — сокращение.
