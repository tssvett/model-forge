## ADR-001 — Объём работы (2026-04-27)

**Контекст.** На момент закрытия задачи `mf-0ew` суммарный объём `diploma/text/*.md` составляет ~23 140 слов (детализация по файлам приведена ниже). При типичной плотности 250 слов на страницу это даёт оценку ~92 страницы. Рекомендуемый ориентир для ВКР бакалавра по направлению 01.03.02 в Самарском университете — 50–70 страниц.

**Распределение объёма по главам:**
- `01-intro.md` — 855 слов;
- `02-review.md` — 4351 слово (наибольшая глава);
- `03-requirements.md` — 1778 слов;
- `04-architecture.md` — 3415 слов;
- `05-implementation.md` — 3382 слова;
- `06-ml-finetuning.md` — 3213 слов;
- `07-experiments.md` — 3726 слов;
- `08-conclusion.md` — 632 слова;
- служебные (титул, аббревиатуры, источники) — ~1790 слов.

**Рассмотренные варианты:**
- (а) оставить текущий объём без сжатия и проверить реальное число страниц после первой сборки `.docx` по шаблону Маршуниной;
- (б) заранее сжать обзор и реализацию на 20–30 % с целью выйти на 65–70 страниц.

**Решение.** Принят вариант (а). Сжатие до фактической сборки `.docx` преждевременно по двум причинам.

Во-первых, реальная плотность слов на страницу при оформлении по шаблону Маршуниной существенно ниже базовых 250: межстрочный интервал 1,5, поля 30/15/20/20 мм, обязательные подписи к рисункам и таблицам, формулы на отдельных строках. Эмпирически такая верстка даёт 180–220 слов на страницу; следовательно, ожидаемое реальное число страниц — 75–85, что лишь незначительно превышает верхнюю границу рекомендации.

Во-вторых, сжатие на 20–30 % с одновременным сохранением логики повествования и ссылочной целостности — это полноценное переписывание четырёх ключевых глав, рискующее ухудшить связность и потребовать перепроверки нумерации рисунков, таблиц и источников. Стоимость риска не оправдана при отсутствии данных о фактическом числе страниц.

**Триггеры для пересмотра.** Решение пересматривается, если после первой компиляции `.docx` (задача отслеживается отдельно) фактический объём превысит 85 страниц без приложений. В этом случае открывается отдельная задача на адресное сжатие наиболее объёмных глав — `02-review.md` (за счёт уплотнения подразделов NeRF/3D Gaussian Splatting/диффузионных подходов) и `07-experiments.md` (за счёт сокращения вводных к каждому эксперименту). Целевое снижение в этом сценарии — до 80 страниц.

**Ответственный за пересмотр.** Жиляев М.И. (после получения первой сборки `.docx`).
